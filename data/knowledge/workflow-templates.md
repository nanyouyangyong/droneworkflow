# 标准工作流模板

## 1. 区域巡检模板

适用场景：电力巡线、光伏电站巡检、建筑外墙检查、管道巡检

### 标准流程
```
开始 → 起飞(30m) → 飞行到巡检起点 → 区域巡检 → 定时拍照(5s) → 电量检查(30%) → [电量充足]继续巡检 / [电量不足]返航 → 降落 → 结束
```

### 关键参数
- 起飞高度：30-50 米（根据巡检对象高度调整）
- 拍照间隔：3-10 秒（光伏电站建议 3 秒，管道建议 5 秒）
- 电量阈值：30%（确保有足够电量返航）
- 巡检速度：3-5 m/s（需要清晰拍照时降低速度）

### 光伏电站巡检特殊要求
- 配备热成像相机，拍照间隔 3 秒
- 飞行高度 20-30 米（确保热成像分辨率）
- 建议在上午 10 点前或下午 3 点后执行（避免强光反射）
- 需要添加热成像拍照节点

### 电力巡线特殊要求
- 飞行高度需高于线路 10 米以上
- 与高压线保持水平距离 ≥ 15 米
- 拍照间隔 5 秒
- 需要沿线路路径设置多个航点

## 2. 紧急搜救模板

适用场景：人员搜救、灾害评估、失踪人员搜索

### 标准流程
```
开始 → 起飞(50m) → 飞行到搜索区域 → 区域巡检(网格搜索) → 录像(start) → 定时拍照(3s) → 电量检查(35%) → [电量充足]继续搜索 / [电量不足]返航 → 录像(stop) → 返航 → 降落 → 结束
```

### 关键参数
- 起飞高度：50-80 米（扩大搜索视野）
- 拍照间隔：2-3 秒（高频拍照避免遗漏）
- 电量阈值：35%（搜救任务需要更高安全余量）
- 搜索模式：网格式搜索，覆盖率 > 95%

### 多机协同搜救
- 使用 parallel_fork 分配搜索区域
- 每架无人机负责一个网格区域
- 建议 2-4 架无人机并行搜索
- 搜索区域之间保留 10% 重叠区域

## 3. 农业植保模板

适用场景：农药喷洒、作物监测、病虫害检测

### 标准流程
```
开始 → 查询天气(当地城市) → [天气适宜]起飞(15m) / [天气不宜]结束 → 飞行到农田起点 → 区域巡检(往返式) → 定时拍照(10s) → 电量检查(30%) → [电量充足]继续作业 / [电量不足]返航 → 降落 → 结束
```

### 关键参数
- 起飞高度：10-15 米（植保作业需低空飞行）
- 飞行速度：3-4 m/s（确保喷洒均匀）
- 航线间距：3-5 米（根据喷幅调整）
- 风速限制：< 3 级（避免药液飘移）

### 注意事项
- 必须先查询天气，风速 > 3 级不可作业
- 避开中午高温时段（药液蒸发快）
- 作业区域周边 100 米内不应有人员
- 建议添加天气条件判断节点

## 4. 设施巡查模板

适用场景：桥梁检查、基站巡查、风电机组检查

### 标准流程
```
开始 → 起飞(50m) → 飞行到设施位置 → 悬停(10s) → 定时拍照(5s) → 录像(start) → 区域巡检(沿边式) → 录像(stop) → 电量检查(30%) → 返航 → 降落 → 结束
```

### 关键参数
- 起飞高度：根据设施高度调整（高于设施 10-20 米）
- 拍照间隔：5 秒
- 巡检模式：沿边式（围绕设施飞行）
- 与设施距离：≥ 5 米

## 5. 测绘航拍模板

适用场景：地形测绘、三维建模、正射影像生成

### 标准流程
```
开始 → 起飞(80m) → 飞行到测绘区域起点 → 区域巡检(往返式) → 定时拍照(2s) → 电量检查(25%) → [电量充足]继续测绘 / [电量不足]标记断点并返航 → 降落 → 结束
```

### 关键参数
- 飞行高度：60-100 米（根据精度要求调整）
- 拍照间隔：2-3 秒（需要 70-80% 航向重叠率）
- 航线间距：根据旁向重叠率计算（通常 60-70%）
- 飞行速度：5-8 m/s
