# 无人机飞行法规与限制

## 中国民航局（CAAC）规定

### 飞行高度限制
- **最大飞行高度**：120 米（AGL，相对地面高度）
- 超过 120 米需向空管部门申请空域
- 机场净空区内严禁飞行

### 禁飞区域
- 机场周边半径 8.4 公里范围
- 军事管理区、国防设施上空
- 核电站、大型水利设施上空
- 政府机关、监狱等敏感区域上空
- 大型活动临时禁飞区

### 飞行时间限制
- 日间飞行（日出后至日落前）
- 夜间飞行需配备防撞灯和导航灯，并获得特殊批准
- 能见度低于 1 公里时不得飞行

### 无人机登记与资质
- 250g 以上无人机需实名登记
- 商业飞行需持有 AOPA 或 CAAC 无人机驾驶执照
- 7kg 以上无人机需申请飞行计划

## 安全飞行距离

| 场景 | 最小安全距离 |
|------|------------|
| 人群聚集区 | 水平距离 ≥ 30 米 |
| 建筑物 | 水平距离 ≥ 10 米 |
| 高压线 | 水平距离 ≥ 15 米 |
| 其他飞行器 | ≥ 500 米 |
| 车辆/道路 | 水平距离 ≥ 10 米 |

## 天气限制

| 天气条件 | 是否可飞 | 备注 |
|---------|---------|------|
| 晴天/多云 | ✓ 可飞 | 最佳飞行条件 |
| 小雨 | ✗ 不可飞 | 电子设备进水风险 |
| 大风（> 6级） | ✗ 不可飞 | 风速 > 10m/s 影响飞行稳定性 |
| 雾霾（能见度 < 1km） | ✗ 不可飞 | 视距外飞行风险 |
| 雷暴 | ✗ 严禁飞行 | 雷击风险极高 |
| 沙尘暴 | ✗ 严禁飞行 | 损坏电机和传感器 |

## 应急处理规范

1. **失控处理**：立即切换至返航模式，如无响应则切断动力
2. **低电量应急**：优先就近安全降落，而非长距离返航
3. **信号丢失**：无人机应自动执行预设的失联返航程序
4. **碰撞风险**：立即悬停或爬升避让，优先保护人员安全
5. **设备故障**：记录故障信息，安全降落后检修
